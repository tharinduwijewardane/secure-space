File Encryptor

A - Available Drives

  • shortcuts to the available volumes on the current device

B - Current Volume Parameters

  • available space / volume size

C - Selected File/Folder

  • long click to select folder
  • short click to select file
  • Selected file or folder can be:
    • Encrypted or Decrypted (according to the state of the selected file) by Encrypt/Decrypt Button (E)
    • Renamed, Opened, Deleted or Wiped by using "menu button" on your device
    • Note: long click on a file leads to the same behavior as using the "menu button" on the selected file/folder

D - Encrypted File

  • encrypted file can be recognized by: .enc extension, purple background and "lock" icon

E - Encrypt/Decrypt Button

  • encrypt or decrypt (according to the currently selected item) file or folder

F - Compression ON/OFF

  • activate/deactivate data compression

G - Directly to the Main Menu

  • go directly to the application Main Menu

H - More Button

  • Selected File/Folder can be Renamed, Deleted, Wiped or Opened (file)

More...

  • only one file or folder can be selected at once - if you want to encrypt more files together, copy them to a folder
  • encryption/decryption process can be interrupted by "Back Button"
  • Wiping - data are physically overwritten before deleting (cannot be undeleted)
