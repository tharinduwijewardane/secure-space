Message Encryptor

A - Copy/Paste Text

  • copy the text from the Text Area C to the system clipboard
  • paste the text from the system clipboard to the Text Area C

B - Encrypt/Decrypt Text

  • encrypt the text in the Text Area C and put the result into theText Area E
  • decrypt the encrypted text in the Text Area E and put in result to the Text Area C

C - Not Encrypted Text Area

  • put the text you want to encrypt here

D - Set Password Button

  • color of this button depends on the current password strength

E - Encrypted Text Area

  • put the text you want to decrypt here

F - More Options Button

  • Load Message from DB: load a previously saved data from the application database
  • Save Message to DB: save data in the Text Area E to the application database for later use
  • Delete Message from DB: delete selected data from the application database

G - Copy/Paste Encrypted Text

  • working just like the Buttons A, but with the encrypted text in the Text Area E

More...

  • Using system clipboard
    • you can use the messages from this application in every Android application you desire to
    • just long click on the chosen "Edit Text" input field in your application and use build-in copy, paste, select, ... system features


  • Incorrect Password
    • there is no way to decrypt any message without correct password
