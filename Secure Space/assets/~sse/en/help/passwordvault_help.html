Password Vault

A - Password Folders

  • Short Click on Folder: open your password item list:
    • Short Click on Item: Show/Hide password
    • Long Click on Item: Show/Edit parameters (name, password, notes and color), Delete/MoveTo item
    • Icons meaning:
      (Copy to Clipboard)
      (Password Generator)
     
     
  • Long Click on Folder: Show/Edit/Delete folder parameters (name, notes and color)
    • Enforce Position (Top Right Corner): (ABC...) means alphabetical order (default) - numbers means absolute position of the edited folder (note: If you choose a position that already exists, the original folder will be replaced on its position by the edited one and the original folder position will be automatically set to default (alphabetical) mode.)
     

B - More Options Button

    note: press "Menu Button" to Show/Hide the Bottom Menu
  • Change Password: change the "master" password for Password Vault
  • Import Vault: import the previously exported Password Vault data from the application Import/Export Directory
  • Export Vault: export (backup) the current Password Vault data to the application Import/Export Directory
  • Notes:
    • application Import/Export Directory can be defined in the Application Settings
    • you can choose between compressed, fully encrypted .pwv file format or unencrypted, editable .xml format

C - New Folder

    note: press "Menu Button" to Show/Hide the Bottom Menu
  • create a new password folder using the same dialog as the Show/Edit/Delete feature

More...

  • Whole vault is encrypted as one object. All attributes (name, password, notes, ...) are stored as secure as the password itself.
  • Forgotten Password
    • there is no way to enter your Password Vault in case you forget your password
    • you can only click on Cancel Button in login (password) dialog and follow "Help me get in!" feature
