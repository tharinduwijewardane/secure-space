Secret Space Encryptor
(version 1.4.6)

  • Password Manager, Message (Text) Encryption and File Encryption integrated in the all-in-one solution.
  • Store and manage all your passwords in one secure place with the Password Vault.
  • Keep your messages, notes and other texts safe from unintended readers with the Message Encryptor.
  • Securely encrypt your private and confidential files or whole folders with the File Encryptor. Wiping (secure delete) feature is included.
  • Everything is encrypted using strong encryption algorithm: AES (Rijndael) 256bit, RC6 256bit, Serpent 256bit, Blowfish 256bit/448bit, Twofish 256bit, GOST 256bit
  • Encryption engine based on the Bouncy Castle Crypto Library (Java) and Crypto++ (C++).
  • Cross Platform (Windows, Linux, Mac OS X, ...) GUI and Command Line Version of the File Encryptor for PC is available on the application home page (download section).
  • This software is Open Source.
